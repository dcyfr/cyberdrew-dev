This is a sample blog post to test the print stylesheet's "Read online" URL feature.

The article element has a data-url attribute with value: https://cyberdrew.dev/blog/test-post

When you print this page (or view print preview), you should see "Read online: https://cyberdrew.dev/blog/test-post" at the very bottom with a border above it.

How It Works

The CSS uses the article:after pseudo-element with content: "Read online: " attr(data-url); to display the URL from the data attribute.
